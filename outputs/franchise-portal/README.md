# Локальный прототип базы знаний франшизы

Открыть: http://localhost:4173

## Демо-доступы

- `owner` / `1234` — владелец, видит все статьи.
- `methodist` / `1234` — методист, видит методические и редакторские материалы.
- `admin` / `1234` — администратор точки, видит операционные материалы и продажи.
- `coach` / `1234` — педагог, видит только методики занятий.

## Что уже есть

- Вход по логину и паролю в демо-режиме.
- Сессия сохраняется после обновления страницы.
- Ролевые доступы к статьям.
- Веб-статьи вместо скачиваемых документов.
- Отдельные режимы базы знаний: обычные статьи и методические уроки.
- Урок может объединять несколько HTML-материалов, например сценарий и чеклист.
- Методики показываются деревом: курс → возраст → урок → материалы.
- Поиск по доступным пользователю материалам.
- Кнопка копирования ссылки на текущую статью.
- Открытие конкретной статьи по ссылке вида `#article=lesson-4-5`.
- Служебные разделы владельца спрятаны в меню справа сверху.
- Отдельная админка: `admin.html`.
- В админке есть дерево материалов, формы редактирования, импорт HTML, добавление картинок и предпросмотр.
- Простой HTML-редактор статей для владельца: только заголовок, раздел и HTML.
- В редакторе есть дерево материалов и блок привязки HTML-материала к уроку.
- Импорт HTML-файла в редактор.
- При импорте HTML удаляются `script`, `style`, служебные теги, inline-стили и опасные обработчики событий.
- Вставка картинок через выбор файла: в HTML остается короткая ссылка `franchise-image:...`, а файл хранится отдельно в браузере.
- Блокировка выделения, копирования, перетаскивания, контекстного меню и горячих клавиш сохранения/печати.
- Защитный экран вместо печати материала.
- Локальный журнал событий.
- Экран прав доступа.

## Следующий слой

- Серверная проверка прав доступа.
- Редактор статей и черновики.
- Версионирование материалов.
